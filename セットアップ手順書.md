| 項目 | 値 |
|------|-----|

| GitHub Pages URL | `https://ichikawa-ux.github.io/property-auto-search` |

## ステップ1: GitHubリポジトリ作成

1. [GitHub](https://github.com/new) を開く
2. Repository name: **`property-auto-search`**（スペースなし・ハイフン）
3. **Private** を選択
4. 「Create repository」
5. PowerShellで以下を実行：

```powershell
cd "C:\Users\ichik\Desktop\不動産物件監視システム"
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/ichikawa-ux/property-auto-search.git
git push -u origin main
```

---

## ステップ2: GitHub Pages の有効化

1. `https://github.com/ichikawa-ux/property-auto-search/settings/pages` を開く
2. Source: **Deploy from a branch**
3. Branch: `main` / フォルダ: `/docs`
4. Save
5. 数分後に `https://ichikawa-ux.github.io/property-auto-search/` が有効になる

---

## ステップ3: Google サービスアカウントの作成

1. [Google Cloud Console](https://console.cloud.google.com/) を開く
2. プロジェクト新規作成（名前は何でもOK）
3. 「APIとサービス」→「ライブラリ」→ **「Google Sheets API」** を有効化
4. 同じく **「Google Drive API」** も有効化
5. 「認証情報」→「認証情報を作成」→「サービスアカウント」
6. 名前を入力して作成
7. 作成したサービスアカウント → 「キー」タブ → 「キーを追加」→「JSON」
8. ダウンロードされたJSONファイルの**中身全体**をコピー（後でGitHub Secretsに貼る）

---

## ステップ4: Googleスプレッドシート作成

1. [Googleスプレッドシート](https://sheets.google.com) で新規作成
2. タイトル: **「物件監視 - 検索条件管理」**（何でもOK）
3. URLの `https://docs.google.com/spreadsheets/d/【ここ】/edit` をコピー → これが **Sheets ID**
4. 「共有」→ ステップ3で作ったサービスアカウントのメールアドレス（`xxxx@xxxx.iam.gserviceaccount.com`）を**編集者**として追加

---

## ステップ5: Googleフォームの自動作成（★重要★）

条件入力はGoogleフォームで行います。スクリプトで自動作成できます。

1. ステップ4のスプレッドシートを開く
2. 「拡張機能」→「Apps Script」
3. 「コードエディタ」に [`google_apps_script/create_form.gs`](google_apps_script/create_form.gs) の内容を**丸ごとコピペ**
4. 1行目の `SPREADSHEET_ID` をステップ4でコピーしたIDに書き換え：

```javascript
var SPREADSHEET_ID = "1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms"; // ← ここを変更
```

5. 「createForm」を選択して「▶ 実行」
6. 権限許可ダイアログが出たら許可
7. ログに表示された **「フォームURL」** をコピー → 担当者全員に共有する

**フォームから入力できる内容：**

| 項目 | 入力方法 |
|------|---------|
| 担当者名 | テキスト入力 |
| メールアドレス | テキスト入力 |
| サイト | チェックボックス（SUUMO / HOME'S / アットホーム） |
| エリア | チェックボックス（東京23区から選択） |
| 家賃上限 | 数字入力（例: 15） |
| 間取り | チェックボックス（1K / 1LDK 等） |
| 築年数上限 | 数字入力（例: 20） |
| 駅徒歩上限 | 数字入力（例: 10） |

> フォームを送信すると自動でスプレッドシートの「検索条件」シートに追加されます。
> 条件を無効にしたい場合はスプレッドシートの「有効」列を `FALSE` にしてください。

## ステップ6: Gmail アプリパスワードの取得

`ichikawa@s-table.co.jp` はGoogle Workspaceのため、そのまま使用できます。

1. `https://myaccount.google.com/apppasswords` を開く
2. アプリ: 「その他」→「不動産監視」と入力
3. 生成された**16文字のパスワード**をコピー

## ステップ7: GitHub Secrets の設定

`https://github.com/ichikawa-ux/property-auto-search/settings/secrets/actions` を開いて、以下の5つを追加：

| Secret名 | 設定する値 |
|----------|-----------|
| `GOOGLE_SHEETS_ID` | ステップ4でコピーしたスプレッドシートID |
| `GOOGLE_CREDENTIALS_JSON` | ステップ3でダウンロードしたJSONの中身全体 |
| `GMAIL_ADDRESS` | `ichikawa@s-table.co.jp` |
| `GMAIL_APP_PASSWORD` | ステップ6で取得した16文字のパスワード |
| `GITHUB_PAGES_URL` | `https://ichikawa-ux.github.io/property-auto-search` |

## ステップ8: Chrome拡張機能のインストール

### アイコン生成（初回のみ）

```powershell
cd "C:\Users\ichik\Desktop\不動産物件監視システム\extension"
pip install Pillow
python create_icons.py
```

### PCのChromeにインストール

1. Chrome → `chrome://extensions/`
2. 右上「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」
4. `extension` フォルダを選択

### AndroidのKiwi Browserにインストール

1. `extension` フォルダを ZIP 圧縮（右クリック → 送る → ZIP）
2. ZIPをAndroidに転送
3. Kiwi Browser → メニュー → 拡張機能 → `+ (from .zip/crx)`

---

## ステップ9: 動作確認

1. Googleフォームから検索条件を1件入力（自分のメールアドレスで）
2. `https://github.com/ichikawa-ux/property-auto-search/actions` を開く
3. 「不動産物件監視」→「Run workflow」で手動実行
4. ログを確認（エラーがないか）
5. 新着物件があれば `ichikawa@s-table.co.jp` にメールが届く

---

## 業者サイトの自動入力設定（ログイン後に実施）

REINS・イタンジ等はログイン必須のため、初回ログイン後に設定が必要です。

1. 業者サイトにログイン → 物件検索ページを開く
2. Chrome DevTools（F12）→ Elements タブ
3. 建物名・住所の入力欄を右クリック →「検証」→ `id` や `name` 属性をメモ
4. [`extension/sites_config.js`](extension/sites_config.js) の該当サイトのセレクタを更新

---

9:00〜19:00（JST）に1時間ごと、平日・休日問わず実行されます。

## トラブルシューティング

| 症状 | 確認箇所 |
|------|---------|
| GitHub Actions が失敗 | Secrets が5つ全部設定されているか確認 |
| メールが届かない | アプリパスワードが正しいか / スパムフォルダを確認 |
| フォームを送信しても条件が反映されない | スプレッドシートの「検索条件」シートに行が追加されているか確認 |
| 物件が取得できない | 各サイトのHTML構造が変更された可能性。GitHubのActionsログを確認 |
| 拡張機能のボタンが出ない | デベロッパーモードON、拡張機能ページでエラーを確認 |
